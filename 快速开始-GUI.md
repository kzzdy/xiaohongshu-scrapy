# 🚀 GUI快速开始指南

## 一分钟上手

### 1. 启动GUI

**Windows用户（最简单）：**
```
双击 "启动GUI.bat" 文件
```

**所有平台：**
```bash
python gui_main.py
```

### 2. 配置Cookie

1. 在GUI中点击"配置"标签页
2. 点击"创建示例.env"按钮
3. 点击"打开.env文件"按钮
4. 将你的Cookie粘贴到 `COOKIES=` 后面
5. 保存文件
6. 回到GUI，点击"重新加载配置"
7. 看到绿色的"✓ 配置加载成功"

### 3. 开始使用

**推荐工作流程（三步走）：**

**第一步：搜索爬取**
- 切换到"🔍 搜索爬取"标签页
- 输入关键词（如：重庆美食）
- 点击"开始搜索（仅保存JSON）"
- JSON文件保存到 `datas/json_datas/`

**第二步：JSON管理器**
- 切换到"📁 JSON管理器"标签页
- 点击"刷新列表"
- 选择JSON文件
- 点击"复制所有链接"

**第三步：笔记爬取**
- 切换到"📝 笔记爬取"标签页
- 粘贴链接（Ctrl+V）
- 勾选"下载图片/视频"
- 点击"开始爬取"

## 如何获取Cookie？

1. 打开小红书网页版：https://www.xiaohongshu.com
2. 登录你的账号
3. 按 `F12` 打开开发者工具
4. 点击 `Network`（网络）标签
5. 刷新页面（按 `F5`）
6. 点击任意一个请求
7. 在右侧找到 `Request Headers`
8. 找到 `Cookie:` 字段
9. 复制整个Cookie值
10. 粘贴到.env文件中

## 功能说明

### 🔍 搜索爬取（第一步）
- 输入搜索关键词
- 设置爬取数量（建议10-50）
- 选择排序方式
- 点击"开始搜索（仅保存JSON）"
- 结果保存为JSON，不下载媒体

### 📁 JSON管理器（第二步）
- 刷新并选择JSON文件
- 查看笔记列表
- 复制所有笔记链接

### 📝 笔记爬取（第三步）
- 粘贴笔记URL（每行一个）
- 选择保存格式（JSON/CSV/Excel）
- 勾选"下载图片/视频"
- 点击"开始爬取"

## 输出文件

所有数据保存在 `datas` 目录：

```
datas/
├── notes/      # 笔记数据
├── users/      # 用户数据
└── search/     # 搜索结果
```

## 常见问题

### Q: 提示"配置加载失败"？
A: 检查Cookie是否正确填写，是否已过期

### Q: 爬取失败？
A: 查看日志页面的详细错误信息

### Q: 如何提高速度？
A: 在.env中调整 `RATE_LIMIT` 和 `MAX_CONCURRENT_DOWNLOADS`
